### Markdown Preview Enhanced

1. 多种特性

- 自动编辑器及预览滑动同步
- 可导出的格式：漂亮的 html、pandoc、电子书、幻灯片、PDF、 PNG、JPEG 。。
- 支持 LaTeX 数学、流程图 / 时序图 以及各种其他种类的图形，嵌入 LaTeX, 渲染 TikZ, Chemfig 等图形
- 支持自定义预览 CSS
- 生成目录。（TOC 生成）
- ...

2. 首先安装插件：点击侧边栏最后一个 Extensions，在搜索框输入 Markdown Preview Enhanced，点击 Install 等待安装完成
3. 打开需要转换的.md 文件，右键选择打开同步预览`Markdown Preview Enhanced:Open Preview to the Side`
4. 生成目录：详细讲解 光标放置想要生成目录的输出位置，Ctrl+Shift+P（MacOS：cmd+shift+p）呼出命令面板，输入 Markdown Preview Enhanced: Create Toc 会生成
5. 然后 Ctrl+s 保存，就会生成目录。==注意：必须要在预览打开的情况下生成操作，保存才能更新。==
6. 输出 html：在右侧的预览界面，右键点击选择 HTML->HTML(offline)。在当前.md 文件的同级目录会生成一个同名.html 文件
